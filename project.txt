담당 역할 및 성과

Spring Boot로 API 서버 개발 및 Vue.js와의 통신 구조 설계

Jenkins + Docker + Kubernetes 조합으로 100% 자동화 배포 파이프라인 구현

PVC 마운트 오류를 해결하기 위해 Longhorn 스크립트 작성

AI 챗봇 연동을 통해 자연어 명령 인식률 90% 이상 확보

운영 중 발생한 DB 접속 실패 및 매핑 NPE 등 장애를 직접 해결하여 안정성 강화

기간: 2024.12 - 2025.03 (4개월)
인원 구성: 5명
사용 기술: Spring Boot, MariaDB, Jenkins, Docker, Kubernetes, Ansible, GitHub

프로젝트 개요
EduLink는 강의 등록, 수강 신청, 과제 제출 등 학습 전반을 관리할 수 있는 웹 기반 학습 관리 시스템입니다.
실제 운영 가능한 시스템을 목표로 개발했으며,
Jenkins 기반 CI/CD 파이프라인과 Docker, Kubernetes, Ansible을 활용하여 무중단 배포 환경을 구축했습니다.
지속성, 안정성, 유연한 운영 환경을 확보하며 인프라 구축과 DevOps 역량을 강화했습니다.

담당 역할 및 성과

Jenkinsfile을 작성해 빌드-테스트-배포 자동화 파이프라인 설계

DockerHub 연동으로 이미지 버전 관리 체계화

readiness probe 미완료 상태에서의 트래픽 전환 문제 해결

Jenkins-Kubernetes 간 환경 변수 누락 문제 구조적으로 해결

Ansible을 활용한 Jenkins 및 Docker 자동 설치 스크립트 작성

블루/그린 배포 전략으로 가용성 100% 유지

프로젝트 개요
Firebase 기반의 실시간 채팅 앱으로, 별도의 서버 없이도 실시간 통신이 가능한 구조를 갖춘 설치형 경량 메신저입니다.

친구 추가, 채팅방 생성, 메시지 송수신 등 주요 기능 개발

중복 채팅방 생성 문제 해결을 위한 ID 기반 고유 키 알고리즘 설계

API 응답 구조 표준화, 예외 처리 체계 구성

실시간 메시지 흐름 제어 및 통신 안정성 확보
